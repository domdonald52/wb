P-Chart Tracer — Help

What this is

A digital ruler for tracing P-charts. You draw the same path through the chart that you'd draw with a pencil on paper; the tool tells you the exit distance on the right-side scale.

Workflow

  1. Load a chart (PNG, JPEG, or PDF — multi-page PDFs supported).
  2. Optional: De-skew. If the chart is slightly rotated (typical for photocopied scans), click ⊞ De-skew, then click 2 points on the same horizontal line, then 2 points on the same vertical line. The image is rotated so the chart's axes line up with the screen. Do this before calibrating — de-skewing clears existing calibration and traces.
  3. Calibrate the distance scale. Click 2+ ticks on the right-hand distance scale (the vertical metre scale). For each click, type the value in metres. Two clicks is enough; more = more accurate fit.
  4. Enter the conditions you're tracing (e.g. PA 2000 / OAT 30 / MTOW / PPD / 5kt HW / +1% slope) in the Conditions field. This becomes the label.
  5. Click "Start new trace", then click points along the chart:
  6. Start at the OAT box's input edge at the right temperature.
  7. Click where you cross each PA line, then move to the next box.
  8. Use Lock H/V for horizontal transfers between boxes (when the trace should go straight horizontally — turning Lock H/V on snaps the next click's Y to the previous click's Y).
  9. Turn Lock H/V off when following oblique lines within a box (e.g. tracing along the weight line, the ops line, the slope line).
  10. End your trace by clicking inside the right-side distance scale area. The tool shows the distance there.
  11. Press Enter or click "Finish trace" to lock in the trace. It appears in the right panel with the calibrated exit distance.

Tips

  • The cursor readout (top-left of the chart) shows the current Y position in metres while tracing — useful while still inside intermediate boxes.
  • Lock H/V toggle: turn on for horizontal moves between boxes, turn off for oblique moves within a box. You'll toggle it multiple times in a typical trace.
  • Ctrl/Cmd+Z undoes the last click in the current trace.
  • You can have multiple traces visible at once — useful for comparing conditions (e.g. PPD vs AT-PD at the same PA/OAT). Each gets a distinct colour.
  • Click the "use" button on a trace in the right panel to make it the active one (so a new edit appends to it).
  • Save session stores all calibration and traces to a JSON file so you can come back later or share it.
  • Download chart with traces exports the chart image with your traces drawn on it — useful for audit / sharing.

What this tool does NOT do

  • It does not generate perf-XXX.js data. It's just a visual aid for cross-checking the app against the chart.
  • It does not simulate the chart's nomogram chain. You do the visual tracing; it reads the distance off the scale.
  • Calibration is Y-only (distance scale). The tool doesn't try to understand the chart's X-axis or boxes.
